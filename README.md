# منصة الألعاب التعليمية - التربية الإسلامية

منصة تعليمية تفاعلية متخصصة في إنتاج ألعاب تعليمية في مادة التربية الإسلامية للصفوف السابع والثامن والتاسع حسب المنهج القطري.

## المميزات

- 🎮 6 أنواع مختلفة من الألعاب التعليمية
- 📚 محتوى متوافق مع المنهج القطري
- 🎨 تصميم احترافي باللغة العربية
- 🔊 أصوات تفاعلية لكل إجراء
- 📱 متوافق مع جميع الأجهزة
- ✍️ إمكانية إنشاء ألعاب جديدة
- 🏆 نظام نقاط وتقييم فوري

## أنواع الألعاب

1. **أسئلة متعددة الخيارات** - اختبر معلوماتك
2. **مطابقة الكلمات** - اربط بين العناصر المتطابقة
3. **ترتيب الجمل** - رتب العناصر بالترتيب الصحيح
4. **ملء الفراغات** - أكمل النص بالكلمات المناسبة
5. **صحيح أو خطأ** - حدد صحة العبارات
6. **لعبة عشوائية** - تحدي عشوائي

## الصفوف الدراسية

- الصف السابع
- الصف الثامن
- الصف التاسع

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla)
- خط Tajawal
- Font Awesome Icons

## التشغيل

1. افتح ملف `index.html` في المتصفح
2. أو استخدم خادم محلي:
```bash
python -m http.server 8000
```
